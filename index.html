TortoiseSI

Windows Explorer Extension to Operate the SCM capabilities of PTC Integrity

TortoiseSI is in an Alpha Release phase. Please see the user documentation for a description of the existing functionality.

Known issues and planned features that are not yet implemented can be found in the GitHub issue list.

TortoiseSI is developed as an Open Source project under the GNU General Public License (GPL-2.0).
